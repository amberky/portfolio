A personal project management tool for organising tasks and projects.
Technologies and Tools
Client-Side
  • Frontend Framework: The user interface is built with Angular 17.
  • Styling: Utilises Tailwind CSS for responsive design, supporting both light and dark themes.
  • State Management: Managed using NgRx in combination with RxJS observables.
Server-Side
  • Backend Framework: Developed with FastAPI in Python, leveraging SwaggerUI for interactive API documentation.
  • Database: Data is stored in a NoSQL MongoDB database.
  • Authentication: Implemented using OAuth 2.0 with Google and JWT tokens.
  • Email Service: Utilises Gmail API to send email to users for account verification and password reset.
Source Control
  • Version Control: Managed with Git and hosted on GitHub.
User Authentication
Support for Google OAuth 2.0 and username/password login
  • The user can sign in using their Google account or by creating a new account with a username and password.
  • The user can also reset their password if they forget it.
  • Account registered with username/password needs to be verified via email.
Project Management
Create, update, and delete projects
  • The user can create a new project, update the project details, and move the project to the trash.
  • The user can also select the project icon (emoji) to represent the project.
Tasks Management
Create, update, and delete tasks
  • Support viewing tasks in board view, calendar view (monthly and weekly), and table view.
  • Set due date, prority, and breakdown tasks into subtasks.
