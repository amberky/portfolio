A personal project management tool for organising tasks and projects.
Technologies and Tools
Client-Side
  • Frontend Framework: The user interface is built with Angular 17.
  • Styling: Utilises Tailwind CSS for responsive design, supporting both light and dark themes.
  • State Management: Managed using NgRx in combination with RxJS observables.
Server-Side
  • Backend Framework: Developed with FastAPI in Python, leveraging SwaggerUI for interactive API documentation.
  • Database: Data is stored in a NoSQL MongoDB database.
  • Authentication: Implemented using OAuth 2.0 with Google and JWT tokens.
  • Email Service: Utilises Gmail API to send email to users for account verification and password reset.
Source Control
  • Version Control: Managed with Git and hosted on GitHub.
